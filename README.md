# QuickDeploy

QuickDeploy is a simple, instant iframe about:blank opener that works Any platform, with simplistic fetures, and troubleshoting and diagnosticts baked right in. 

## What Dose It Work With?

We work with many sites that alow for a full page embed, however, if the site you want to cloak dose not have that option, there are some workarounds.

One common workaround is using a proxy like Ultravilot (https://github.com/titaniumnetwork-dev/Ultraviolet) Or Scramjet (https://github.com/MercuryWorkshop/scramjet), and by properley deploying it using a deployment (https://github.com/titaniumnetwork-dev/Ultraviolet-App) for Scramjet
